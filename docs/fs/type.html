PsiFS: File System

File Types

File Types

The *PsiFSMap command is used to control the mapping of file types between RISC OS and SIBO or EPOC. The default mappings used by PsiFS are contained in the file !PsiFS.Configure, which may be modified to add extra mappings or to change the existing ones.

Note that it is not possible to change the file type of an EPOC file from RISC OS; the file type information is part of the file's contents.
