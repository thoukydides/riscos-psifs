RISC OS owner attributes are mapped to and from SIBO or EPOC read only (or not writable) and system attributes as follows:
RISC OSSIBO or EPOC
read onlysystem
   /    
  R/  Yes 
 W /

L  /  Yes 
L R/  YesYes
LW /   Yes
LWR/   Yes

RISC OS public attributes are mapped to and from SIBO or EPOC archive (or modified) and hidden attributes as follows:
RISC OSSIBO or EPOC
archivehidden
   /    
   / r Yes
   /w YesYes
   /wrYes 

This mapping preserves all SIBO or EPOC attributes except for the system attribute of directories, which is converted to read only due to limitations of RISC OS filesystems.
