PsiFS: *Commands

Serial Link

The following *commands provided by the PsiFS module allow the serial block driver to be configured and controlled. It should not normally be necessary to use these *commands directly; the desktop filer allow the settings to be changed more simply.
  • *PsiFSDisable
  • *PsiFSDriver
  • *PsiFSEnable
  • *PsiFSListDrivers
  • *PsiFSStatus

Filing System

The following *commands relate directly to the operation of the PsiFS filing system.
  • *Drive
  • *Free
  • *NameDisc and *NameDisk
  • *PsiFS
  • *PsiFSMap

Miscellaneous

The following additional *commands provided by the PsiFS module offer general usefully functions.
  • *PsiFSSIS
  • *PsiFSTar
