PsiFS: *Commands

*PsiFSStatus
*PsiFSStatus
Display the status of the block driver

Syntax:*PsiFSStatus

Parameters:None

Use: *PsiFSStatus displays the status of the block driver and any active remote link or printer connection.

Example:*PsiFSStatus
Block driver 'InternalPC' active on port 0.
0 bytes received, 0 bytes transmitted.
0 valid frames and 0 invalid frames received, including 0 retries.
0 frames transmitted, including 0 retries.
Not connected to a remote device.
The remote link may be disabled, or the settings could be incorrect.

Related commands:*PsiFSDisable, *PsiFSDriver, *PsiFSEnable, *PsiFSListDrivers
