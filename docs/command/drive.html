PsiFS: *Commands

*Drive
*Drive
Sets the current drive

Syntax:*Drive drive

Parameters:drive the letter of the disc drive, from A to Z

Use: *Drive sets the current drive if NoDir is set. Otherwise, *Drive has no meaning. The command is provided for compatibility with early versions of ADFS.

Example:*Drive C

Related commands:*Dir, *NoDir
