PsiFS: *Commands

*PsiFS
*PsiFS
Selects the Psion Filing System as the current filing system

Syntax:*PsiFS

Parameters:None

Use: *PsiFS selects the Psion Filing System as the filing system for subsequent operations. Remember that it is not necessary to switch filing systems if you use the full pathnames of objects. For example, you can refer to ADFS objects (on a local harddisc, say) when PsiFS is the current filing system.

Example:*PsiFS

Related commands:*ADFS, *Net, *RAM, *ResourceFS, *SCSI
