PsiFS: *Commands

*PsiFSListDrivers
*PsiFSListDrivers
List the available serial block drivers

Syntax:*PsiFSListDrivers [-verbose]

Parameters:-verbose display more information about each driver, including the version number and the baud rates supported

Use: *PsiFSListDrivers lists the serial block drivers available for use by PsiFS.

Example:*PsiFSListDrivers
Driver     Information                     Manufacturer
---------- ------------------------------- -------------------------------
Internal   Internal (Acorn wiring)         The Serial Port
InternalPC Internal (PC wiring)            The Serial Port
