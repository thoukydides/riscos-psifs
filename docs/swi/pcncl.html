PsiFS: SWI Calls

PsiFS_PrintJobCancel
PsiFS_PrintJobCancel
(SWI &520D4)
Cancel a print job

On entry:R0 =handle of print job

On exit:Registers preserved

Interrupts:Interrupt status is undefined
Fast interrupts are enabled

Processor mode:Processor is in SVC mode

Re-entrancy:SWI is not re-entrant

Use: Cancel a print job. This is only appropriate for EPOC devices.

If this is used while the print job is still being received, the original source is cancelled. Otherwise, this just discards the data that has been queued.


Related SWIs:PsiFS_PrintJobPoll, PsiFS_PrintJobData

Related vectors:None
