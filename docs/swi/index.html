PsiFS: SWI Calls

SWI Calls

The following SWIs are implemented by the PsiFS module. These are mainly intended for use by the desktop filer, but may be used by other clients.
  • PsiFS_Register
  • PsiFS_Unregister
  • PsiFS_Set
  • PsiFS_Get
  • PsiFS_Mode
  • PsiFS_AsyncStart
  • PsiFS_AsyncEnd
  • PsiFS_AsyncPoll
  • PsiFS_AsyncControl
  • PsiFS_FileOp
  • PsiFS_InterceptClaim
  • PsiFS_InterceptRelease
  • PsiFS_InterceptPoll
  • PsiFS_InterceptControl
  • PsiFS_CheckUID
  • PsiFS_ClipboardCopy
  • PsiFS_ClipboardPaste
  • PsiFS_GetTranslationTable
  • PsiFS_PrintJobPoll
  • PsiFS_PrintJobData
  • PsiFS_PrintJobCancel

A comprehensive OSLib interface to all of the PsiFS SWIs and WIMP messages is included as part of the source code release.
