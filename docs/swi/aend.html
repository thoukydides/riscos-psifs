PsiFS: SWI Calls

PsiFS_AsyncEnd
PsiFS_AsyncEnd
(SWI &520C6)
End an asynchronous remote operation

On entry:R0 =the previously allocated handle for this operation

On exit:Registers preserved

Interrupts:Interrupt status is undefined
Fast interrupts are enabled

Processor mode:Processor is in SVC mode

Re-entrancy:SWI is not re-entrant

Use: End a previously started asynchronous remote operation. If the operation is still in progress, then this may either terminate the operation or allow it to complete depending on the operation being performed.

Related SWIs:PsiFS_AsyncStart, PsiFS_AsyncPoll, PsiFS_AsyncControl

Related vectors:None
