PsiFS: Getting Started

Printing from EPOC Devices

Printer via PC

EPOC users as advised to print via the RISC OS printer drivers rather than use the printer mirror. The following instructions explain how to use the printer mirror.

Default Printer

The default printer on EPOC devices is configured from the Control panel. This may be opened from the System screen by clicking on the Control panel toolbar icon, selecting the Control panel... option from the Tools menu, or by pressing Ctrl+S. Double-click on the Printer icon to open the Standard printer setup dialogue box.

There are two important options:

  • Print to - This should be set to a suitable value for the type of printer actually being used. It must not be set to Printer via PC, even though output will be redirected via the RISC OS computer.
  • Print via - This must be set to Serial port.

Next, click on the Settings... icon or press Ctrl+S to open the Serial port settings dialogue box. The baud rate must be set to the same value as configured within PsiFS. The other settings should be set as shown below:

Click on the OK icon or press Enter to confirm the serial port settings and close the dialogue box, then click on the OK icon or press Enter to confirm the standard printer setup and close the dialogue box. The Control panel may be closed either by clicking outside its window or by pressing Escape.

The default printer should now be configured correctly for use via the PsiFS printer mirror.

Printing

Ideally, having set the default printer, it would be sufficient to simply select the Print... option from any application. However, some applications maintain their own individual printer options, either globally or for individual files. Hence, it is necessary to verify the settings each time a file is printed.

The following instructions describe how to print from Word; the procedure for other applications should be similar.

The two machines should be connected, and the printer mirror enabled, if not already done. Word should be started and a suitable document opened, such as the Welcome to Series 5 file that comes with Psion Series 5 machines.

Open the Print dialogue box either by selecting Print... from the Printing sub-menu of the File menu, or by pressing Ctrl+P. Next, open the Printer setup dialogue box either by clicking on the Printer... icon or by pressing Ctrl+P. Set the To and Via options as described above, i.e. to the appropriate printer type and Serial port respectively.

Open the Serial port settings dialogue box either by clicking on the Settings... icon or by presing Ctrl+S. Verify that the options are set as described above; they should be correct, but it is worth checking.

Click on the OK icon or press Enter to confirm the serial port settings and close the dialogue box, then click on the OK icon or press Enter to confirm the standard printer setup and close the dialogue box. The document may now be printed either by clicking on the Print icon or by pressing Enter.
