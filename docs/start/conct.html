Connecting

Click on the PsiFS iconbar icon with Select, or select Remote link from the iconbar icon menu, to enable the remote link. The idle icon will be replaced by the remote link icon to indicate that a connection is being attempted:

From the System screen of the SIBO or EPOC device enable the remote link and ensure that the correct baud rate has been set:

  • On SIBO devices the link is configured using the Remote Link or Communications option from the Special menu, or by pressing Psion+L or Acorn+L.
  • On EPOC devices the link is configured using the Remote link... option from the Tools menu, or by pressing Ctrl+L.

After a few seconds, a drive icon should appear on the iconbar for each remote drive that is accessible. A different icon is used for SIBO or EPOC drives to indicate the type of device connected:
SIBO drive EPOC drive

The drive icons can be used just like any other desktop filer. For example, to open a directory viewer for the root directory of any drive simply click on the icon with Select.

The link may be disconnected at any time.
