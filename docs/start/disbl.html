Disabling the Printer Mirror

To disable the printer mirror click on the iconbar icon with Select or Adjust, or select Disconnect from the iconbar icon menu. The idle icon will be displayed to indicate that the serial port is no longer being used:

No action is required on the SIBO or EPOC device; the printer mirror is only active while an application is printing via the serial port.
