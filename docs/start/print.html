Printing via the Remote Link

PsiFS supports printing from EPOC devices via the remote link. This uses the normal RISC OS printer drivers, so any printer with a suitable RISC OS driver can be used, even if no driver exists for EPOC.

This is provided as an alternative to the printer mirror, which allows the SIBO or EPOC printer drivers to be used.

Default Printer

The default printer on EPOC devices is configured from the Control panel. This may be opened from the System screen by clicking on the Control panel toolbar icon, selecting the Control panel... option from the Tools menu, or by pressing Ctrl+S. Double-click on the Printer icon to open the Standard printer setup dialogue box.

The Print to option should be set to Printer via PC. There are no other settings for this printer driver. Click on the OK icon or press Enter to confirm the standard printer setup and close the dialogue box. The Control panel may be closed either by clicking outside its window or by pressing Escape.

The default printer should now be configured correctly for use via PsiFS and the remote link.

Printing

Ideally, having set the default printer, it would be sufficient to simply select the Print... option from any application. However, some applications maintain their own individual printer options, either globally or for individual files. Hence, it is necessary to verify the settings each time a file is printed.

The following instructions describe how to print from Word; the procedure for other applications should be similar.

The two machines should be connected, and the remote link connected, if not already done. Word should be started and a suitable document opened, such as the Welcome to Series 5 file that comes with Psion Series 5 machines.

Open the Print dialogue box either by selecting Print... from the Printing sub-menu of the File menu, or by pressing Ctrl+P. Next, open the Printer setup dialogue box either by clicking on the Printer... icon or by pressing Ctrl+P. Set the To option as described above, i.e. to Printer via PC, and then click on the OK icon or press Enter to confirm the standard printer setup and close the dialogue box. The document may now be printed either by clicking on the Print icon or by pressing Enter.

By default this will open a window on the RISC OS computer showing the status of the print job, and offering the following options:

  • Print - Start printing, if a suitable printer driver is loaded.
  • Preview - Display a preview of the print job. This also allows the pages to be exported in a form suitable for loading into the Draw application.
  • Cancel - Discard the print job.
However, PsiFS can be configured to automatically start printing or display the preview when a print job starts.
