Installing EPOC SIS Files

The PsiFS file format converter interface supports the installation of SIS files on EPOC devices. To use this feature, ensure that there is an active remote link and then drag a SIS file to one of the PsiFS iconbar icons. In the file conversion window that opens, select the Install SIS file option, and click on Install.

PsiFS will automatically ensure that the Add/remove icon is present on the control panel. It will then copy the SIS file and start the installation process. Follow the prompts displayed on the EPOC device to complete the installation.

Double-clicking Files

It is also possible to start the installation of SIS files by double-clicking on their icons within filer windows. However, depending on the configuration of PsiFS it may be necessary to hold down the Left Alt key to force PsiFS to intercept the file.
