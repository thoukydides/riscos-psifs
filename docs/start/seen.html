Installing File Format Converters

The simplest way to use a third party file format converter with PsiFS is to ensure that it has been seen by the filer before PsiFS is loaded. Alternatively, the file format converter can be placed within the !PsiFS.Converters directory to ensure that it is always available.

See the documentation supplied with individual file format converters for more details or any additional requirements. The file format converters section of the FAQ lists the most common third party tools.

Once file format converters have been installed, they may be easily used to perform conversions.

!ChangeFSI

PsiFS includes a simple file format converter that uses !ChangeFSI to convert common graphics formats (GIF, PCX, BMP, PBM, JPEG, TIFF, and Sprite files) to either Sprite or JPEG files with a selectable number of bits per pixel.

No installation is required to use this file format converter. If necessary, PsiFS automatically searches for a copy of !ChangeFSI, but if this fails the message Please open a directory containing !ChangeFSI and then repeat the operation is displayed. In this event simply open a filer window for the appropriate directory.

!PsiFS

PsiFS includes another simple file format converter that uses the PsiFS module to extract the contents of tar and SIS files. No installation is required to use this file format converter.

Please note that long filenames will only be preserved if !Scrap is located on a drive that supports long filenames.
