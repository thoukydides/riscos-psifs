PsiFS: Compiling

Compiling

Several additional tools and libraries are required to compile PsiFS.

Once the tools and libraries have been correctly installed it should be sufficient to run the MakeFile to build both !PsiFS.Resources.PsiFS and !PsiFS.Resources.!RunImage. Similarly, the documenation can be built by running Docs.BuildDocs.
