PsiFS: Iconbar Icons

Iconbar Icon

The PsiFS iconbar icon changes appearance depending on the current status. The possible icons are:
Idle Printer mirror Remote link SIBO drive EPOC drive

Note that when the remote link is enabled, and PsiFS is connected to a remote SIBO or EPOC device, there may be more than one iconbar icon; one for each accessible drive on the remote device.

Each icon behaves differently when clicked on using the mouse with each of the three mouse buttons. However, the behaviour has been selected to be intuitive and efficient to use.

Dragging a file to any of the iconbar icons will open a window allowing a file conversion to be performed.
