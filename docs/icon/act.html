PsiFS: Iconbar Icons

Remote Link

Remote Link Iconbar Icon

The remote link iconbar icon is displayed when PsiFS is attempting to connect to a remote SIBO or EPOC device using the Psion Link Protocol. If a connection is successfully made then the remote link icon is replaced by an SIBO or EPOC drive icon (as appropriate) for each accessible remote drive.

Dragging a file to the iconbar icon will open a window allowing a file conversion to be performed.

Select

Clicking on the iconbar icon with Select has no effect.

This has the same effect as selecting Disconnect from the iconbar icon menu.

Menu

Clicking on the iconbar icon with Menu displays the following menu:
PsiFS
Info
Help
Disconnect
Quit

Adjust

Clicking on the iconbar icon with Adjust disables the remote link. The idle icon is displayed to indicate the new state.

This has the same effect as selecting Disconnect from the iconbar icon menu.
