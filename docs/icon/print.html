PsiFS: Iconbar Icons

Printer Mirror

Printer Mirror Iconbar Icon

The printer mirror iconbar icon is displayed when PsiFS is copying all characters received over the serial link to the printer. This allows a SIBO or EPOC device to print to a parallel printer without needing a special cable.

The text under the icon changes to indicate the number of bytes received over the serial link. This can be useful to confirm that the printer mirror is operating as expected.

More information is available on enabling and disabling the printer mirror. This includes details specific to printing from SIBO and EPOC devices.

Dragging a file to the iconbar icon will open a window allowing a file conversion to be performed.

Select

Clicking on the iconbar icon with Select has no effect.

This has the same effect as selecting Disconnect from the iconbar icon menu.

Menu

Clicking on the iconbar icon with Menu displays the following menu:
PsiFS
Info
Help
Disconnect
Quit

Adjust

Clicking on the iconbar icon with Adjust disables the printer mirror. The idle icon is displayed to indicate the new state.

This has the same effect as selecting Disconnect from the iconbar icon menu.
