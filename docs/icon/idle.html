PsiFS: Iconbar Icons

Idle

Idle Iconbar Icon

The idle iconbar icon is displayed when PsiFS has been disabled. In this state the serial port is available for use by other applications.

Dragging a file to the iconbar icon will open a window allowing a file conversion to be performed.

Select

Clicking on the iconbar icon with Select initiates a connection to a remote SIBO or EPOC device using the Psion Link Protocol. The remote link icon is displayed to indicate the new state.

This has the same effect as selecting Remote link from the iconbar icon menu.

Menu

Clicking on the iconbar icon with Menu displays the following menu:
PsiFS
Info
Help
Remote link
Printer mirror
Configure...
Quit

Adjust

Clicking on the iconbar icon with Adjust enables the printer mirror. The printer mirror icon is displayed to indicate the new state.

This has the same effect as selecting Printer mirror from the iconbar icon menu.
