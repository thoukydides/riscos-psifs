PsiFS: Compatibility

Host Computer

PsiFS should work on any computer running RISC OS 3.1 or higher.

It has been successfully used on:

  • Acorn A410/1 fitted with RISC OS 3.1.
  • Acorn A4 fitted with RISC OS 3.1.
  • Acorn A5000 fitted with RISC OS 3.11.
  • Acorn RiscPC 600/700/J233 fitted with an ARM610/710/StrongARM/Kinetic and RISC OS 3.5/3.6/3.7/4.0.

Early Acorn Archimedes computers had faulty serial ports that required cables wired differently to operate correctly. The Psion 3-Link and simple RS-232 serial cable will not work unmodified with these machines, but the Acorn A-Link should work without problems.

Remote Computer

PsiFS should work with any computer running either SIBO (Psion Series 3/3a/3c/3mx, Psion Workabout, Psion Siena, Acorn PocketBook I/II or Xemplar PocketBook III) or EPOC (Psion Series 5/5mx/7, Psion Netbook, Psion Revo, Psion Revo Plus, Geofox-One, Ericsson MC218, Oregon Scientific Osaris, Philips Ilium Accent or Philips Synergy) connected via a serial link.

It has been successfully used with:

  • Psion Series 3 fitted with ROM version 1.80(F), 256kB RAM.
  • Psion Series 3a fitted with ROM version 3.40(F), 2MB RAM.
  • Psion Series 3c fitted with 2MB RAM.
  • Psion Series 3mx.
  • Acorn PocketBook II fitted with ROM version 1.20(F).
  • Psion Series 5 release 1 fitted with ROM version 1.00(113)/1.01(114)/1.01(145) and 8MB RAM.
  • Psion Series 5mx fitted with ROM version 1.05(250).
  • Psion Revo.
  • Psion Revo Plus.
  • Psion Netbook.
  • Psion Workabout.
  • Geofox-One fitted with 16MB RAM.
  • Ericsson MC218.
  • Oregon Scientific Osaris.

Unfortunately, some EPOC based mobile phones use a modified version of Psion Link Protocol that is not currently supported by PsiFS. This includes the Ericsson R380, Nokia 9210 and Nokia 9290.

Serial Link

PsiFS should work over any serial link that can be used with PsiWin (Psion 3-Link, Acorn A-Link or a simple RS-232 serial cable for more recent machines).

It has been successfully used with the following cables:

  • Psion 3-Link cable (does not work unmodified on early Acorn Archimedes machines).
  • Acorn A-Link cable.
  • Psion Series 5 serial cable (does not work unmodified on early Acorn Archimedes machines).

It has been successfully used with the following serial ports:

  • Internal serial port.
  • The Serial Port dual serial port card.
  • Atomwide triple serial port card.
  • Intelligent Interfaces dual serial port card.

Although it has not been tested, PsiFS should also work via an IrDA infrared link if both machines are fitted with suitable interfaces and protocol stacks.
