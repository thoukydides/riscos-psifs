PsiFS: Known Problems

Problems With This Version

See the frequently asked questions for details of other problems that may be experienced.
  • Under RISC OS 4 enabling the printer mirror without an active printer connected can result in an infinite error loop due to the new parallel port error detection. There probably isn't much that can be done about this.
  • RISC OS attributes such as LR/, LW/ or LWR/ result in the EPOC system attribute being set. Unfortunately, this makes it impossible to delete the file from an EPOC device.
  • Corrupt tar files can result in confusing error messages.
  • Some print jobs, notably those from Sheet, have spurious lines plotted.

Missing Features

  • It may be useful to allow just part of a disc to be backed up.
