PsiFS: Menu Entries

Iconbar Icon Menus

Each PsiFS iconbar icon has a different menu associated with it. The possible menu entries are:
  • Backup...
  • Configure...
  • Disconnect
  • Free...
  • Help
  • Info
  • Name disc
  • Printer mirror
  • Quit
  • Remote link
