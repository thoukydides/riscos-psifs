Name disc

Name disc

Moving the mouse pointer to the right over the Name disc sub-menu icon opens a window allowing the disc name to be changed. This is equivalent to *NameDisc.

A disc name must contain at least two, and at most ten, printing characters. The name must not contain any of the following characters:

$ & % @ \ ^ : . # * " |
