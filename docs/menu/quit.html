PsiFS: Menu Entries

Quit

Quit

Selecting the Quit menu entry quits both the PsiFS filer and the file system module. This first disconnects any active connection to a remote SIBO or EPOC device.

Moving the mouse pointer to the right over the Quit sub-menu icon displays the following menu:
Quit
Filer
All

Quit > Filer

Selecting the Filer menu entry, from the Quit menu, quits the PsiFS filer. This leaves the PsiFS file system module running. Any active connection to a remote SIBO or EPOC device remains available.

Quit > All

Selecting the All menu entry, from the Quit menu, quits both the PsiFS filer and the file system module. This first disconnects any active connection to a remote SIBO or EPOC device.

This has the same effect as selecting the Quit menu entry.
