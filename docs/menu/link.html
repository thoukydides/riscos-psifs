Remote Link

Selecting the Remote Link menu entry initiates a connection to a remote SIBO or EPOC device using the Psion Link Protocol with the configured serial block driver and baud rate. The remote link icon is displayed to indicate the new state.

This has the same effect as clicking on the idle icon with Select.
