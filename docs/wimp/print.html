PsiFS: WIMP Messages

Message_PsifsPrint

Message_PsifsPrint (&520C1)

R1+20preferred action
R1+24initial position x coordinate (-1 for automatic)
R1+28initial position y coordinate (-1 for automatic)
R1+32filename, zero-terminated
This message is broadcast by a task that wants the PsiFS filer to process a page from a print job.

The possible actions are:
ActionDescription
0open control window
1display preview
2start printing

If PsiFS accepts the print job then it will take a copy of the file and acknowledge the message.
