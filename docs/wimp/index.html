PsiFS: WIMP Messages

WIMP Messages

The following WIMP messages are supported by the PsiFS filer. These are mainly intended for internal use, but may be used by other clients.
  • Message_PsifsAsyncStart
  • Message_PsifsPrint

A comprehensive OSLib interface to all of the PsiFS SWIs and WIMP messages is included as part of the source code release.
