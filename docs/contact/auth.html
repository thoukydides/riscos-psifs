Contacting the Author

I would be interested in hearing of any bugs or other unexpected features in PsiFS, and will endeavour to correct any such problems in future releases. Suggestions for improvements are also most welcome. Implementation depends upon practicability and on how much spare time I have.

If in any communication you make specific reference to the program code then please refer only to the software as released. It would also be helpful if you could include all of the information requested by the fault report form.
