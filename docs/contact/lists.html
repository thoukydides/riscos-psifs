Mailing Lists

A mailing list has been created for announcements and discussions concerning PsiFS. The mailing list is controlled by sending email to the following addresses:
riscos-psifs@yahoogroups.com
Send a message to all subscribers to the mailing list.
riscos-psifs-subscribe@yahoogroups.com
Subscribe to the list.
riscos-psifs-unsubscribe@yahoogroups.com
Unsubscribe from the list.
There is also a web based interface available at http://groups.yahoo.com/group/riscos-psifs that provides control over delivery options and gives access to an archive of previous postings.
